RESTful Routes

这一节，我们通过 TDD 设置三个新的路由，遵循 RESTful 最佳实践。

Endpoint HTTP Method CRUD Method Result
/users GET READ get all users
/users/:id GET READ get single user
/users POST CREATE add a user

我们要实现：

  1. 写一个测试
  2. 运行测试，看着它失败（红色）
  3. 编写足够的代码以使测试通过（绿色）
  4. 重构（如有必要）
