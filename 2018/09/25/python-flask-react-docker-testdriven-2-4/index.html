Flask Debug Toolbar

本节讲解 Flask调试工具栏 。

Flask Debug Toolbar 是一个 Flask 扩展，可以帮助您调试应用程序。它在视图中添加了一个调试工具栏，它提供有关 HTTP 标头，请求变量，配置设置以及呈现特定视图所需的 SQLAlchemy 查询数的信息。您可以使用此信息来查找视图呈现中的瓶颈。
