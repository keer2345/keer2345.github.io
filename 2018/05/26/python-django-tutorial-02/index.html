Serializers 序列化

Serializers序列化是Django Restful framework很重要的一個地方，主要功能是将Python结构序列化为其他格式，例如我们常用的JSON。

在上一节中，我们建立了musics模型，这里我们来将其进行序列化。

在musics中新增serializers.py：

serializers.SerializerMethodField()

测试API

什么是Restful API

  • GET获取资源
  • PUT更新资源
  • DELETE删除资源
  • POST新增资源
  • PATCH更新资源的部分内容

测试工具

测试API的工具很多，这里我们选择Postman，使用方法很简单，在Chrome中安装Postman应用。

POST

添加数据也很方便，输入地址http://127.0.0.1:8000/api/music/，选择`POST`类型，然后在*body*的地方填入`song`和`singer`的值，点击*send*，就是提交了。
